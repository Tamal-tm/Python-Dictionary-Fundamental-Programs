# Python-Dictionary-Fundamental-Programs

A. Overview

This collection covers basic to intermediate Python dictionary programs that help understand key concepts like creation, access, modification, iteration, and common operations.

B. Topics Covered

1. Creating dictionaries using {} or dict()

2. Accessing values using keys and .get()

3. Adding & updating items dynamically

4. Removing elements using pop(), del, and clear()

Looping through keys, values, and key-value pairs

Checking existence of keys using in

Nested dictionaries and their access

Dictionary methods: .items(), .keys(), .values(), .update()

Practical programs like:

--Counting word frequency

--Merging two dictionaries

--Reversing key-value pairs

--Finding maximum/minimum values

--Sorting dictionaries by key or value
